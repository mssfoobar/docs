# Development

- IAN is a module that provides in-app notifications to users, using RESTful.
- The backend is written in Go, using [chi](https://github.com/go-chi/chi) framework for routing
- Database using PostgreSQL: package [sqlx](https://github.com/jmoiron/sqlx)
- Logger using aoh-logger, based on [zap](https://go.uber.org/zap)
