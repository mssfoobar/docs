# Overview

This section highlight how to deploy PTMGR in a Kubernetes cluster.

## Architecture

<div>
    <div style={{ marginBottom: "2rem", maxWidth: "300px" }}>
        <IdealImage img={architecture} />
    </div>
</div>

PTMGR has the following service dependencies:
- Postgres
    - Required by PTMGR to store token expiration date.
- IAMS (Keycloak + AAS)
    - Required by PTMGR for API request authentication and storing FCM tokens in IAMS user profile.
