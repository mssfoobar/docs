# Overview

This section highlight how to deploy UNH in a Kubernetes cluster.

## Architecture

<div className="centerize">
    <div style={{ marginBottom: "2rem", maxWidth: "700px" }}>
        <IdealImage img={architecture} />
    </div>
</div>

UNH has the following service dependencies:
- Postgres
    - Required by UNH-APP for storing notification templates, distribution lists and channels configuration.
- IAMS (Keycloak + AAS)
    - Required by UNH-APP for API request authentication and recipient contact details resolution.
    - Required by UNH-WEB for login authentication and distribution list creation.
