# Deploy Playback namespace

-   Navigate to manifest/playback folder
-   Fullfil and correct variables in `playback-configmap.yml`

:::info
Configration: [Config](../configuration/playback.mdx)
:::

-   Create a playback configmap

```bash
kubectl apply -f playback-configmap.yml
```

-   Create a playback deployment

```bash
kubectl apply -f playback-deployment.yml
```

-   Create a playback service

```bash
kubectl apply -f playback-service.yaml
```

-   Create an ingress

```bash
kubectl apply -f playback-ingress.yml
```
