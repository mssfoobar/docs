# Features
The UCS module is a real-time communication system that allow users to chat, call and share files with each other in a
virtual conference room.

-   **Video Conferencing** - Room based video conferencing to share video, audio, chat and files primarily for
    team-sized collaboration in the course of fulfilling a task.
-   **Room Management APIs** - Provide HTTP APIs to manage rooms and participants.
-   **Client SDK** - A client library to join, leave, chat, video/audio streaming in a room.
-   **Security** - WebRTC based secure communication between peers.
