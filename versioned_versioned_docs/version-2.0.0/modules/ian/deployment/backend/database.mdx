# Database

- Postgres installation: [Postgres](https://www.postgresql.org/download/)
- Initialize database by docker-compose:
    - Navigate to repository directory
    - Start postgres
        ```bash
        docker-compose up database -d
        ```
    - Now postgres is running, can connect to postgres via [docker network](https://docs.docker.com/engine/network/)
