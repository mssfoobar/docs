Easy As Pie

Developer experience takes a front-seat in AGIL Ops Hub. Our goal is to ensure the project is enjoyable to use so developers will enjoy writing and maintaining the projects derived from AGIL Ops Hub.

Powered by Svelte

Svelte is a radical new approach to building user interfaces. Svelte shifts that work into a compile step that happens when you build your app.

Instead of using techniques like virtual DOM diffing, Svelte writes code that surgically updates the DOM when the state of your app changes.

Powered by Tailwind

Tailwind is a utility-first CSS framework with mobile-first responsive classes, state variants, easy extensibility, a tiny package size from removing unused CSS on production build, and much, much more.

Powered by GraphQL

GraphQL allows us to query for exactly what we need, straight in the front-end. With GraphQL's powerful introspection, we can easily write and test type-safe queries using graphql-tag and graphql-codegen.

Powered by Hasura

Replace simple back-end services with Hasura - create your schema and get your GraphQL API instantly. Merge larger services into your GraphQL API with Hasura's actions feature.

Identity & Access Management

Identity and access management (provided with Keycloak) - giving us many features including OpenID Connect, Single Sign-On, LDAP, and Active Directory as well as social logins.

Ready for Testing

Write end-to-end tests with Playwright and manage them with Allure for test report generation and viewing. Manage your code coverage and unit tests with Vitest and C8 for nodejs applications.
