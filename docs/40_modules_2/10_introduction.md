# Introduction

The modules section contains all the available modules developed by AOH, catered for C2 systems in particular. These modules offers a seamless integration with the `Web-Base`, a frontend base template built with `Svelte` and `SvelteKit`, that aims to accelerate the initial setup and development for frontend projects.

This set of documentation assumes that you are using `Web-Base` as a starting point.
