# Features
The Workflow Engine(WFE) is an eSOP system developed for AOH projects. It offers the following features:

-   **Workflow Orchestration** - Manage business process with workflow automation and orchestration.
-   **Workflow Designer** - Design business process models using a user-friendly web interface.
-   **Workflow Management APIs** - Use APIs to manage and control workflow executions.
-   **Developer SDK** - Create custom business activities with a comprehensive SDK.
-   **Monitoring and Analytics** - Monitor and analyze workflow executions in real-time.
-   **Scalable and High Availability** - Developed with a focus on scalability and high availability architecture.
-   **BPMN 2.0 Compliant** - Compliant with the BPMN 2.0 specification.
