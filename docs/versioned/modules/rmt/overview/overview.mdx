# Overview

The Resource Management and Tasking (RMT) module provides a service for managing resources and task assignments within an organization. It enables efficient allocation, and tracking resources while facilitating effective task management.

### Key Features

## Features

- **Resource Management**: Manage resources and resource assignment tracking
- **Task Management**: Manage tasks and task assignment to resources

## Technical Details

- Built with Go
- Uses PostgreSQL for data persistence
- RESTful API design using Chi router
- GIS: Uses the `GIS` module to manage resources and tasks locations
