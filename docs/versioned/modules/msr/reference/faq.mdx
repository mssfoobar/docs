# Frequently Asked Questions

This document answers some frequently asked questions about the MSR (Multi-Session Replay) module.

**Q: What is the performance impact of the MSR recording agent?**

A: The MSR recording agent is designed to be lightweight and have a minimal impact on the performance of your application. However, the exact impact will depend on the complexity of your application and the recording mode you are using.

**Q: How is the recorded data secured?**

A: All recorded data is encrypted both in transit and at rest. Access to the replay UI is controlled by IAMs, ensuring that only authorized users can view recorded sessions.

**Q: Can I record sessions from any web application?**

A: The MSR recording agent is compatible with most modern web applications built with frameworks like React, Angular, and Vue. However, some complex or unusual applications may require additional configuration.
