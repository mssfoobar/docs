# Pre-requisites

## Access and Tools

-   [Node.js](https://nodejs.org/en/) (version ^20)
-   [git](https://git-scm.com/downloads) (version ^2.47)
-   Access to the source code of the [IAN GitHub Repository](https://github.com/mssfoobar/ian)

## Assumptions

All installation instructions assume that:

-   You are using either the [web base](/docs//modules/web-base/overview/overview.md) or running IAN in "local development"
-   Already have set up [access configuration](/docs/modules/get-started/access.mdx).
-   Have the API Endpoint Access of Keycloak, SSE Client service and IAN backend from either of the following environments:
    -   Locally on Development Environment. Using [dev-containers](/docs/modules/get-started/local-development.mdx) with UNH and IAMS module enabled is recommended.
    -   Deployed on Production or Staging Environment.
-   You are using the Windows Operating System.
